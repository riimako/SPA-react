## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/riimako/SPA-react

# Access
$ cd SPA-react

# Install dependencies
$ npm

# Run the project
$ npm run dev

# The server will initialize in the <http://localhost:5173>
```
